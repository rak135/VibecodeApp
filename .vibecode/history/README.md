# VibecodeApp Change History

`.vibecode/history/README.md` is committed as the history policy.

`.vibecode/history/*.md` may be committed only when it contains durable project truth.
`.vibecode/runs/*` is runtime output and remains ignored.

History summaries are compressed project memory, not raw logs.

Suggested sections for a summary:

- Task
- Changed files
- Behavior changed
- Tests run
- Decisions
- Follow-up
